Session 어노테이션 연습

[[${session.url}]]- 세션 스콥의 url 애트리뷰트 값
[[${url}]] - 스콥 지정하지 않은 url 애트리뷰트 값
url 이라는 이름의 애트리뷰트값을 가져올 때 스콥 범위가 작은 순서로 찾습니다.

세션에 저장된 애트리뷰트는 브라우저가 종료되거나 삭제 될때까지 사용할 수 있습니다.

  • [[${session.userDTO.name}]]
  • [[${session.userDTO.email}]]
  • [[${session.userDTO.tel}]]
  • [[${session.userDTO.password}]]
  • [[${memberDTO.name}]] : request 스콥에서 가져온 memberDTO 애트리뷰트
  • [[${memberDTO.email}]]
  • [[${memberDTO.tel}]]
  • [[${memberDTO.password}]]

@SessionAttribute 는 세션 스콥에서 값을 가져오는 동작을 합니다. @ModelAttribute 는 요청에서 값을 가져와 DTO 에 저장하고, 그 값을 그대로 템플릿(타임리프 html)에 전달 합니다. => request 스콥입니다.
